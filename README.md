#YouTube-Analysis 

This project focuses on analyzing YouTube video data using Python to extract meaningful insights about audience behavior, engagement, and content performance. It utilizes datasets in CSV and JSON formats and applies various data analysis and visualization techniques.

🚀 Features:
1. Text Sentiment Analysis
Performed sentiment analysis on YouTube comments using TextBlob
Generated word clouds to visualize positive and negative sentiments
2. Emoji Analysis
Extracted and analyzed emojis from comments using the emoji library
Visualized emoji frequency using bar charts
3. Data Integration
Imported multiple CSV and JSON files
Merged datasets into a single dataframe for unified analysis
4. Category-wise Likes Analysis
Identified categories with the highest likes
Represented insights using box plots
5. Audience Engagement Analysis
Evaluated engagement metrics such as:
Like rate
Dislike rate
Comment rate
Visualized using box plots
6. Likes vs Dislikes Analysis
Explored the relationship between likes and dislikes
Used regression plots for visualization
7. Correlation Heatmap
Created a heatmap to identify relationships between:
Views
Likes
Dislikes
8. Trending Channels Analysis
Identified the channel with the highest number of trending videos
